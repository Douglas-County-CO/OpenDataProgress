    Greetings, Developer

    Aside from being free to use, public data is different from private data in two major ways. The first is that public data does not include any personally identifiable information, and private data sometimes does. The second is that public data is always secondary data, meaning it is collected by the government for the purposes of government operations (primary data), but then when re-purposed by data consumers it becomes secondary data. This means that, quite often, data consumers are limited in their capacity to request changes to the data that meet their needs. In some scenarios, there is an open feedback loop, where data consumers can request enhancements to the formatting or content of the data. At Douglas County you can suggest a data set at: https://data.douglas.co.us/nominate or contacting us directly at opendata@douglas.co.us. If you make something great with our data we would also love to hear about it!

    Socrata Platform

    This is the data platform that has been selected by Douglas County to store all of our tabular data. You can access our portal at: https://data.douglas.co.us/


    Platform provides:

    • An API platform, SDK's, and app developer examples to get started. Access to the the Socrata Open Data API (SODA) http://dev.socrata.com/
    • Ability to export any dataset via CSV, JSON, RDF, RSS, XML, and OData formats.
    • Visualize any dataset as column, bar, pie, donut, line, area, and bubble.

    Socrata Google Maps

    If you are using some of our location data and are wanting to put a location map on your website take a look at this library.


    How to Get/Use It:

    • Download from Github here: https://github.com/mikegiddens/Socrata-GoogleMaps
    • Explore data using our oData url's in the Export for our datasets
    • See how we are using it in action: http://www.douglas.co.us/community/senior-adult-services/senior-resources

    Socrata Visualizations

    If you are looking for an easy way to generate simple graphs, pie charts, and comboboxes, check out our Javascript library to quickly render graphic components based on Socrata as the data source.


    How to Get/Use It:

    • Download from Github here: https://github.com/mikegiddens/socrata-visualizations
    • Hook it up to any of our datasets or any other Socrata data provider.
    • Check out our publishing plan for live examples.

    CARTODB

    Location intelligence and visualization engine that empowers you to effortlessly transform data into insights.


    How to Get/Use It:

    • Create an account to gain free access at www.cartodb.com, upload data, style it and then publish an iFrame or shareable link.
    • Tell data stories and perform complex spatial analyses using multiple layers in a single map frame; customize queries and css styles all in one access point.

    Microsoft Power BI

    Power BI is a powerful business analytics tool from Microsoft that can be run locally or in the cloud. Office365 users should be able to find and install PowerBI in the suite of applications provided by their Office365 subscription.


    How to Get/Use It:

    • Download Power BI here: https://powerbi.microsoft.com/en-us/
    • Explore data using our oData url's in the Export for our datasets
    • Check out some great examples from the State data portal. http://gocode.colorado.gov/data/data-exploration-examples/
    • Create stunning interactive reports and embed visuals directly on your website.
